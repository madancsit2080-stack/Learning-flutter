## ⚙️ Installation & Setup

### 1) Clone the repository

```bash
git clone https://github.com/madancsit2080-stack/Learning-flutter.git
cd Learning-flutter
```

### 2) Open a specific Flutter project folder

```bash
cd <project-folder>
flutter pub get
flutter run
```

### 3) Verify Flutter installation

```bash
flutter doctor
```
